# Portfolio Mehdi Benabid

Portfolio développeur web junior moderne, sombre et responsive, construit avec React, Vite, Tailwind CSS et Framer Motion.

## Fonctionnalités

- Hero animé avec glow, particules et portrait professionnel
- Navbar fixe avec scroll smooth et menu hamburger mobile
- Section projets avec cards modernes, stack technique et liens externes
- Section compétences avec cartes animées et barres de progression
- Section contact avec informations directes et formulaire stylé
- Design responsive mobile, tablette et desktop
- Animations Framer Motion sobres et professionnelles

## Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React

## Installation

```bash
npm install
```

## Lancer le projet

```bash
npm run dev
```

Le site sera disponible sur :

```txt
http://127.0.0.1:5173
```

## Build production

```bash
npm run build
```
